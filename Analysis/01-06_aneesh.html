2. LOADING IN THE DATA

We see from the overall dataset, there’s definitely a cyclical aspect to the dataset, but overall a steady level of visitors. Each year, there appears to be a very slight upward trend. This is hard to verify because of a strong increase in visitors in the middle of 2016. This strong discontinuity cannot be modeled, so the data used will be cut off at july 2016 or later.

genre_totals <- genre_sum %>% 
  mutate(yr_m = format(visit_date,"%Y-%m")) %>%
  filter(yr_m %in% c('2017-04','2017-03')) %>%
  group_by(air_genre_name) %>%

store_alloc <- merged %>% 
  mutate(yr_m = format(visit_date,"%Y-%m")) %>%
  filter(yr_m %in% c('2017-04','2017-03')) %>%
  group_by(air_store_id, air_genre_name) %>% 
  summarize(store_visitors =sum(visitors)) %>% 
  left_join(genre_totals, by=c('air_genre_name'='air_genre_name')) %>% 
  mutate(pct=store_visitors / visitors) %>% 
  select(air_store_id, air_genre_name, pct) %>% 
  arrange(air_genre_name)
